## Check 3: Filter helper defined correctly

**Function at line 265-269:**
```python
def _filter_transient_paths(paths):
    """Drop paths whose basename starts with `_staging_` — these are transient
    atomic-deposit filenames that exist only between write and move and are not
    deliverables. Per LESSONS 2026-05-18 strike-4 entry."""
    return {p for p in paths if not os.path.basename(p).startswith("_staging_")}
```

Uses `os.path.basename(p)` — filters on basename only, not the full path. A path like `knowledge/decisions/_staging_foo.md` correctly has basename `_staging_foo.md` which triggers the filter, while `knowledge/staging/real-file.md` would not be filtered (basename is `real-file.md`).

**Result: PASS**
